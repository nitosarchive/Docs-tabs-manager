# DOCS Tabs Manager

## Description
This is an improved version of the [Chrome Tabs Manager Extension](https://developer.chrome.com/docs/extensions/get-started/tutorial/popup-tabs-manager) from the chrome extension tutorial.
It is speifically for docs. I made it because I wanted to learn and I saw how the project could be improved. 

## Features
Group and ungroup chrome and mdn doc tabs[only for now], I plan on making improvements either now or upon a later date.
Can go to site upon clicking on one of the listed sites.


## How to use locally
First download this repo to your device
Go to chrome chrome://extensions/
On top of the screen click "Load unpacked" and select the folder that the extension resides in.
Pin extension and have fun!
